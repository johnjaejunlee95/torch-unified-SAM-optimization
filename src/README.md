## Source Code Documentation

This directory contains the core implementation of the **Unified SAM approaches**.  
It includes modularized codes for model architectures, data loaders, and various SAM-based optimizers.

## Directory Structure

```text
src/
├── data/        # Dataloaders (CIFAR, TinyImageNet)
├── model/       # Backbone architectures (ResNet, WideResNet, etc.)
├── sam_optim/   # SAM and its variants implementation
└── utils/       # Training/Evaluation loops and helper functions
```
